# FormSecure

Mobile-first React web app for verified ticket resale with tiered phase browsing and secondary market valuation estimates.

## Features

- **Tiered ticket phases** — Browse Early Bird, Main Wave, and Die-Hard VIP tiers per event
- **Market metrics** — Original face value, current escrow-verified resale price, and predictive gate value
- **Escrow checkout** — Secure purchase flow via slide-up escrow sheet
- **Die-Hard waitlist** — Join tier-specific waitlists when phases sell out

## Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS v4

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
